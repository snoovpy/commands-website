Bot Commands

Type any command in chat to use it  ·  prefix every command with !

Basic
!hello
Bot greets you by name.
!ping
Check if the bot is alive. Returns Pong!
!lurk
Let chat know you're lurking in the shadows.
!commands
Shows the link to this page.
Stream Info
!title
Shows the current stream title.
!game
Shows the current game or category.
!uptime
How long the stream has been live, or how long it's been offline.
!followers
Shows the total follower count of the channel.
!followage
How long you've been following the channel.
!accountage
How old your Twitch account is. Use !accountage @user for others.
!schedule
Shows the streaming schedule.
!shoutout @user
Shouts out another streamer with a link to their channel.
Fun
!dice
Rolls a random dice (1–6).
!coinflip
Flips a coin. Heads or tails.
!8ball <question>
Ask the magic 8ball anything.
!rps rock/paper/scissors
Play rock paper scissors against the bot.
!quote
Shows a random saved quote. Use !quote <number> for a specific one.
Socials
!twitter  /  !x
Links to snoovpy's X (Twitter) page.
!discord
Links to the Discord server.
Mod Only 🔒
!addcmd !name response
Adds a new custom command on the fly.
Mod only
!removecmd !name
Removes a custom command.
Mod only

Saves a new quote.
Mod only
!quote remove <number>
Removes a quote by number.
Mod only
